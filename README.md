# QRKot - Благотворительный фонд поддержки котиков

## Описание проекта

QRKot — это API-приложение для благотворительного фонда поддержки котиков. 
Фонд собирает пожертвования на различные цели: корм для бездомных кошек, 
медицинское обслуживание, организацию приютов и другую помощь кошачьей популяции.

**Версия 2.0** — добавлена аутентификация и авторизация пользователей через FastAPI Users.

### Основные возможности

- **Управление целевыми проектами**: создание, редактирование, удаление и просмотр 
  проектов с определённой суммой сбора (только для суперпользователей).
- **Пожертвования**: авторизованные пользователи могут делать пожертвования с комментариями.
- **Автоматическое инвестирование**: пожертвования автоматически распределяются 
  по открытым проектам в порядке их создания (FIFO).
- **Аутентификация и авторизация**: регистрация и вход через JWT токены.
- **Разграничение прав доступа**:
  - Анонимные пользователи: просмотр проектов, регистрация
  - Авторизованные пользователи: создание пожертвований, просмотр своих пожертвований
  - Суперпользователи: полный доступ к управлению проектами и просмотру всех пожертвований
- **Прозрачность**: каждый проект и пожертвование имеют полную историю 
  с датами создания и закрытия.

### Технологии

- **FastAPI** — современный веб-фреймворк для Python
- **FastAPI Users** — аутентификация и авторизация
- **SQLAlchemy** — ORM для работы с базами данных
- **Alembic** — миграции базы данных
- **Pydantic** — валидация данных
- **SQLite** — база данных (для разработки)
- **Uvicorn** — ASGI-сервер
- **JWT** — JSON Web Tokens для аутентификации
